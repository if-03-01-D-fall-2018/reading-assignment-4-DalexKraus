The essential technologies that made the World Wide Web a success were URL, HTTP and HTML


URL:

The UniformResourceLocators is used to address specific resources on a web server through the internet using a domain or host name.

HTTP:

The HyperTextTransferProtocol is used to send .html files, which contain the content of the website, from the server to the client.

HTML:

The HyperTextMarkupLanguage is a computer language, which is used as a type of 'skeleton' for a website, which is interpreted by the web client, the browser.


How are resources (html pages, images, etc) addressed in the web?

To locate or address specific pages, images or videos in a website, you have to use something what's called an URL.
URL's (Uniform Resource Locators) were defined in RFC 1738 in 1994 by Sir Tim Berners-Lee, the inventor of the World Wide Web.
Most web browsers display the URL of a web page above the page in an address bar.

General components of an URL

A typical URL could have the form http://www.example.com/index.html, which indicates a protocol (http), a hostname (www.example.com), and a file name (index.html).


Web Sever

What is a web server?

Web server refers to server software, or hardware dedicated to running said software, that can serve contents to the World Wide Web.
A web server processes incoming network requests over HTTP and several other related protocols.

What does a web server?

The primary function of a web server is to store, process and deliver web pages to clients.
The communication between client and server takes place using the Hypertext Transfer Protocol (HTTP).
Pages delivered are most frequently HTML documents, which may include images, style sheets and scripts in addition to the text content.

How can content be stored on a web server?

Web servers are able to map the path component of a Uniform Resource Locator (URL) into:
- A local file system resource (for static requests)
- An internal or external program name (for dynamic requests)
For a static request the URL path specified by the client is relative to the web server's root directory!

Today websites are also able to support serverside scripts or applications such as php, ruby or node.js.


Web Client

What is a web client?

A web browser (commonly referred to as a browser) is a software application for accessing information on the World Wide Web.
Each individual web page, image, and video is identified by a distinct URL, enabling browsers to retrieve and display them on the user's device.

CSS

Selectors

Colors

They can be specified by names (e.g. red) or by hexadecimal values. (e.g. #ffac4d)
They are specified by the rule 'color'.
Same can be used for the rule 'background-color'!


Text alignment

Text can be aligned by using the rule 'text-alignment' and all acceptable values are 'right, left, center and justify'


Text decoration

Text can be decorated using underlines, overlines and strike-troughs.
Using the rule 'text-decoration' the decoration of a text can be set.


Font weights

A font can be marked as heavier, using the 'weight' tag.


Text transformations

Texts can be transformed by using the 'text-transform' rule.
Acceptable values are 'uppercase', 'lowercase' and 'capitalize'.

Lists

Unordered Lists use with the ul tag. Every item in a list has to be marked using the tag li! Unordered Lists don't have a numeration for each entry, for this you have to use the ol tag.
The icon of a bullet mark can be changed using a css-rule: list-style-type Possible values are disc, circle, square, none.

Ordered Lists This list gives every entry a specific index number. This type of number can be changed using the attribute type.
Possible values for types are:

  • 1 - Every entry is given a number
  • A - Every entry is given an uppercase letter
  • a - every entry is given a lowercase letter
  • I - every entry is given a uppercase roman letter
  • i - every entry is given a lowercase roman letter

Tables

An HTML table is defined with the table tag.
Each table row is defined with the tr tag.
A table header is defined with the th tag.
By default, table headings are bold and centered. A table data/cell is defined with the td tag.
Borders can be set using the css rule border.
Example: border: 1px solid black.
If you want the borders to collapse into one border, add the CSS border-collapse property and set it to collapse.
